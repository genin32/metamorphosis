Description

This page serves as a repository for Metamorphosis.

The mod is available to download at https://www.nexusmods.com/fallout3/mods/23658

Metamorphosis is not compatible with Tale of Two Wastelands.

Requirements

  • Fallout Script Extender v1.3 b2
  • Command Extender
  • Fallout Anniversary Patcher
  • NVAC
